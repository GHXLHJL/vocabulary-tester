# 自动单词检测小工具 Spec

## Why
用户需要一个类似 Excel 的单词背诵检测工具。在输入中文释义后，系统应能自动判断正误，并提供直观的颜色和图标反馈（正确为绿色✅，错误为红色❌）。同时，工具需要支持后续添加新单词，以便用户不断扩充自己的词库。

## What Changes
- 创建一个基于 Web 的单页面应用（使用原生 HTML/CSS/JS，轻量且易于运行）。
- 设计一个三列的表格界面：【英文单词】、【你的答案】、【检测结果】。
- 实现输入框实时或失焦后的正误判断逻辑。
- 实现“添加新单词”功能，允许用户录入新的英文单词及对应的正确中文释义。
- 使用浏览器本地存储（LocalStorage）保存单词数据和用户的答题记录，确保刷新页面不丢失。

## Impact
- Affected specs: 无（全新项目）
- Affected code: 在当前目录下新建前端文件，不影响现有的安全学习笔记 markdown 文件。

## ADDED Requirements
### Requirement: 单词自测核心功能
系统应提供一个类似表格的界面供用户测试单词。

#### Scenario: 单词输入正确
- **WHEN** 用户在“你的答案”列输入正确的中文释义（如 ascribe -> 将...归因于）
- **THEN** 该单元格/行背景变绿，并在“检测结果”列显示“✅ 正确”

#### Scenario: 单词输入错误
- **WHEN** 用户在“你的答案”列输入错误的中文释义
- **THEN** 该单元格/行背景变红，并在“检测结果”列显示“❌ 错误”

### Requirement: 词库管理功能
系统应允许用户扩展词库。

#### Scenario: 添加新单词
- **WHEN** 用户在“添加单词”区域填写新的英文单词和对应的正确释义并点击保存
- **THEN** 新单词被追加到测试列表中，并且数据被持久化保存到浏览器的本地存储中
