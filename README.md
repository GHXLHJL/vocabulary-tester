# 安服工程师实习生学习资料总览

这份学习资料是根据安服工程师实习生的招聘要求重新规划和整理的，涵盖了Web安全、渗透测试、系统安全、工具使用、代码能力和面试准备等核心内容。

---

## 📚 文档结构说明

### 1. [Web安全基础与漏洞分析](./01_Web安全基础与漏洞分析.md)
**核心内容**：
- OWASP Top 10 (2025最新版) 详细解读
- 核心漏洞详解（SQL注入、XSS、RCE、文件包含等）
- 业务逻辑漏洞分析
- 代码层面的"好"与"坏"示例
- 流量分析与应急响应

**学习目标**：理解Web安全核心漏洞原理，掌握漏洞挖掘和防御方法。

### 2. [渗透测试实战指南](./02_渗透测试实战指南.md)
**核心内容**：
- 渗透测试标准流程 (PTES 7步法)
- 信息收集核心技术
- 漏洞挖掘方法
- 应急响应流程
- 渗透测试报告撰写

**学习目标**：掌握渗透测试的完整流程，能够独立开展渗透工作。

### 3. [系统与网络安全配置](./03_系统与网络安全配置.md)
**核心内容**：
- 安全配置与基线加固
- 常见安全设备认知 (WAF、IDS/IPS、HIDS/EDR等)
- Windows常用命令 (CMD/PowerShell)
- Linux常用命令 (Bash)
- SQL Server基础操作
- 数据安全技术 (加密、访问控制、数据脱敏)
- 防火墙与WAF配置

**学习目标**：熟悉系统安全配置，掌握常用命令和安全设备知识。

### 4. [安全工具与代码能力](./04_安全工具与代码能力.md)
**核心内容**：
- 基础安全工具使用指南 (Nmap、Sqlmap、Burp Suite等)
- CTF常用工具 (逆向工程、密码学、流量分析、隐写、取证)
- Git版本控制工具
- 代码编写能力 (Python、Bash、PowerShell脚本)
- 后端开发基础 (Flask、Django、Spring Boot)
- YAML语法与插件开发
- AI大模型与深度学习基础

**学习目标**：熟练使用主流安全工具，具备一定的代码编写能力。

### 5. [攻防对抗与ATT&CK框架](./05_攻防对抗与ATT&CK框架.md)
**核心内容**：
- Shellcode免杀技术
- Web攻防对抗技术
- 后渗透技术
- ATT&CK攻击链完整内容
- 防御策略
- 推荐学习资源

**学习目标**：理解攻击者的思路和防御方法，掌握ATT&CK攻击链。

### 6. [面试准备与职业发展](./06_面试准备与职业发展.md)
**核心内容**：
- 突击学习策略 (3-5天计划)
- 技术面试实战话术
- 综合面试实战话术
- 实战案例分析 (SQL注入、XSS、文件上传、越权等)
- 面试技巧与注意事项
- 软件开发与测试基础 (作为面试知识点)
- 技术文档编写、团队协作 (作为面试知识点)
- 实习面试自测清单
- 推荐学习资源

**学习目标**：掌握面试技巧，能够自信地回答面试问题，了解职业发展路径。

## 🎯 学习路径建议

### 阶段1：基础夯实 (1-2周)
1. 学习《Web安全基础与漏洞分析》，理解OWASP Top 10和核心漏洞原理。
2. 学习《系统与网络安全配置》，掌握Linux和Windows常用命令。
3. 搭建本地靶场环境（DVWA、Pikachu），进行漏洞复现练习。

### 阶段2：工具掌握 (1-2周)
1. 学习《安全工具与代码能力》，掌握Nmap、Sqlmap、Burp Suite的使用。
2. 学习《攻防对抗与ATT&CK框架》，理解攻击者的思路和防御方法。
3. 编写简单的自动化脚本（Python端口扫描、目录扫描等）。
4. 在线平台练习（Hack The Box、TryHackMe）。

### 阶段3：实战提升 (2-3周)
1. 学习《渗透测试实战指南》，掌握渗透测试的完整流程。
2. 独立完成2-3个完整的渗透测试项目。
3. 编写渗透测试报告，练习报告撰写能力。

### 阶段4：面试准备 (1周)
1. 学习《面试准备与职业发展》，背诵常见问题和标准答案。
2. 整理项目经验，准备详细的案例说明。
3. 进行模拟面试，练习表达和应对。

## 📖 推荐学习资源

### 在线学习平台
- **PortSwigger Academy**: Burp Suite官方出的实验室，免费且权威。
- **DVWA (Damn Vulnerable Web Application)**: 经典的漏洞练习环境。
- **CTFHub**: 国内比较好的Web基础练习平台。
- **Hack The Box**: 在线渗透测试平台。

### 书籍推荐
- **《Web安全深度剖析》**: 深入理解Web安全原理。
- **《Python黑帽子：黑客与渗透测试编程之道》**: Python安全编程入门。
- **《Metasploit渗透测试指南》**: Metasploit框架使用指南。

### 视频教程
- **B站**: 搜索"SQL注入演示"、"XSS攻击演示"等关键词。
- **YouTube**: HackerSploit, IppSec等安全频道。

## 🎓 学习建议

1. **理论与实践结合**：不要只看书，要多动手实践，搭建靶场环境进行练习。
2. **循序渐进**：从基础漏洞开始，逐步深入复杂漏洞和渗透技术。
3. **记录笔记**：学习过程中记录重要知识点、命令和技巧，方便复习。
4. **参与社区**：加入安全社区，参与讨论，分享学习心得。
5. **持续学习**：安全领域更新很快，要保持学习的热情和习惯。
